# front-test

## Project setup
```
npm install
```

### Create file .env from .env.example and add API KEY var value
```
VUE_APP_AI_KEY=API_KEY
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
